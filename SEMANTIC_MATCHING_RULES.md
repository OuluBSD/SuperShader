# Semantic Matching Rules

This document describes the semantic matching algorithms used in data flow validation.

## Position-related
Terms: position, pos, fragpos, worldpos, Position, FragPos

## Normal-related
Terms: normal, norm, n, Normal, N

## Color-related
Terms: color, col, albedo, diffuse, Color, Diffuse

## TexCoord-related
Terms: uv, texcoords, tex_coords, TexCoord, UV

## Light-related
Terms: light, lightpos, lightdir, lightPos, lightDir
